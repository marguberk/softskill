## 🌐 Live Demo
**🚀 Официальный сайт:** https://berkinbayev.kz

## 📖 Описание

MaroAI - это современная интеллектуальная обучающая платформа для развития soft skills, созданная с использованием FastAPI, React и искусственного интеллекта. Платформа предлагает интерактивные курсы, персонализированное обучение и систему геймификации.

## 🛠️ Технический стек

### Backend
- **FastAPI** - современный, быстрый веб-фреймворк
- **Python 3.12** - мощный язык программирования
- **SQLite** - легковесная база данных
- **Pydantic** - валидация данных
- **Uvicorn** - ASGI сервер

### Frontend
- **React 18** - современная библиотека для UI
- **TypeScript** - типизированный JavaScript
- **Vite** - быстрый инструмент сборки
- **Tailwind CSS** - utility-first CSS фреймворк
- **Lucide React** - современные иконки

### Deployment & DevOps
- **Ubuntu 24.04** - операционная система сервера
- **Nginx** - веб-сервер и reverse proxy
- **Let's Encrypt** - SSL сертификаты
- **systemd** - управление сервисами
- **GitHub Actions** - CI/CD (планируется)

## 🚀 Быстрый старт

### Локальная разработка

```bash
# Клонирование репозитория
git clone https://github.com/marguberk/softskill.git
cd softskill

# Установка зависимостей backend
pip install -r requirements.txt

# Запуск backend
python main.py

# Установка зависимостей frontend (в новом терминале)
cd frontend
npm install

# Запуск frontend
npm run dev
```

### Production деплой

Подробная документация по деплойменту: [DEPLOYMENT.md](./DEPLOYMENT.md)

```bash
# Быстрый деплой на Ubuntu 24.04
./setup-server.sh

# Настройка SSL
./setup-server-ssl.sh your-domain.com
```

## 🔧 API Документация

### Основные endpoints:
- **Health Check:** `GET /api/v1/health`
- **Пользователи:** `POST /api/v1/users/register`
